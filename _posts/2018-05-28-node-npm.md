# NPM

> 노드에서 사용하는 패키지 관리자

## npm install [options]

- `-g` : 전역으로 설치함
- `--save` : 프로젝트의 `node_modules` 폴더에 설치하고 `package.json` 파일의 `dependencies` 항목을 업데이트함
- `--save-dev` : 프로젝트의 `node_modules` 폴더에 설치하고 `package.json` 파일의 `devDependencies` 항목을 업데이트함
